About MQLess

MQLess is a router in front of AWS Lambda. When you send a message to an actor, MQLess queue the message to the actor's mailbox and activate the actor on AWS Lambda. When the actor is done processing the message, MQLess send the response back to the caller and process the next message on the mailbox.
