Get started

Simple

MQLess doesn’t have hierarchy or supervisors. MQLess is not a cluster, it is the router in front of the AWS Lambda.

Avaiable

MQLess actor always exists virtually, you don’t need to create or stop actors. Simply, just send them a message.

Language agnostic

MQLess is language agnostic, you can write your actors in any language supported by AWS Lambda.

About MQLess

MQLess is a router in front of AWS Lambda. When you send a message to an actor, MQLess queue the message to each actor mailbox and activate the actor on AWS Lambda. When the actor is done processing the message, MQLess send the response back to the caller and process the next message on the mailbox.
